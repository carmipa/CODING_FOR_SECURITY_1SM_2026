## Processador de Dados de Cadastro

**Objetivo:** Praticar entrada de dados (`input`), armazenamento em variáveis, operações matemáticas simples e manipulação de strings.

### Enunciado

Você foi contratado para criar um script inicial que padroniza o cadastro de novos usuários em um sistema. O programa deve:

1. Solicitar o primeiro nome e o sobrenome do usuário separadamente.
2. Solicitar o ano de nascimento e o ano atual.
3. Calcular a idade aproximada do usuário.

### Saída Esperada

- Exibir o nome completo em letras **MAIÚSCULAS**.
- Exibir o nome completo em letras **minúsculas**.
- Exibir uma mensagem de boas-vindas confirmando a idade calculada.
  - *Exemplo: "Olá, Joaozinho da silva, você tem aproximadamente 31 anos"*.
