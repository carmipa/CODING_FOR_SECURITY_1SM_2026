Instruções e Regras
• Conteúdo:
o Formato: Avaliação individual.
o Tempo: Vocês terão 1h45 para completar o desafio.
o Não permitido: Comunicação com colegas não é permitida, Uso de IAs e 
Celular.
o Atividades: Atividade Prática.
o Entrega: Submeter o arquivo com seu nome-sobrenome.py na atividade 
do teams.
O Desafio de Hoje: Agenda de Contatos
• Título: O Desafio: Sua Própria Agenda de Contatos!
• Missão: Criar um sistema de gerenciamento de contatos que permita ao 
usuário:
o Adicionar um Contato: Pedir o nome e o telefone e salvar.
o Listar Contatos: Mostrar todos os contatos salvos.
o Buscar um Contato: Procurar um contato pelo nome e mostrar o 
telefone.
o Sair: Encerrar o programa.
• Requisitos obrigatórios:
o Loop Principal: O programa deve rodar em um while e só parar quando 
o usuário escolher "Sair".
o Estrutura de Dados: Use um dicionário para armazenar os contatos. A 
chave pode ser o nome e o valor, o telefone.
o Tratamento de Erros: Use try-except para garantir que o telefone 
digitado seja um número. Se o usuário digitar um texto, o programa deve 
avisar do erro e continuar rodando, sem quebrar! E encerrar o programa 
com o finally.
(obs: uso do raise é ideal para capturar o erro específico.)
